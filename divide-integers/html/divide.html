Divide two integers

Aufgabe

Erstellen Sie eine Datei mit dem Namen divide.py und Implementieren Sie eine Python-Funktion divide(dividend, divisor), die zwei ganze Zahlen dividiert, ohne die Operatoren /, * oder % zu verwenden. Verwenden Sie stattdessen wiederholte Subtraktion, um die Division zu durchzuführen. Das Ergebnis soll zur Null hin abgeschnitten werden (d.h. 8.9 → 8 und -8.9 → -8).

Hinweise
  • Verwenden Sie while Schleifen, um das Dividieren durch wiederholtes Subtrahieren zu implementieren.
  • Denken Sie daran, auf das Vorzeichen zu achten (positiv/negativ) und es auf das Ergebnis anzuwenden.
  • Denken Sie daran, eine Division durch Null zu vermeiden oder eine exception auszulösen.
  • Für weitere Informationen zu exception können Sie die folgenden Links besuchen:
    • www.w3schools.com/python/ref_keyword_raise.asp
    • docs.python.org/3/reference/simple_stmts.html#raise
