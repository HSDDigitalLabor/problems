{% include "file.html" %} Implementieren Sie eine Python-Funktion divide(dividend, divisor), die zwei ganze Zahlen dividiert, ohne die Operatoren /, * oder % zu verwenden. Verwenden Sie stattdessen wiederholte Subtraktion, um die Division zu simulieren. Das Ergebnis soll zur Null hin abgeschnitten werden (d.h. 8.9 → 8 und -8.9 → -8). Falls das Ergebnis außerhalb des Bereichs eines 32-Bit-Integer liegt (-231 bis 231 - 1), geben Sie den Grenzwert zurück.

Hinweise
  • Verwenden Sie while Schleifen, um das Dividieren durch wiederholtes Subtrahieren zu implementieren.
  • Denken Sie daran, auf das Vorzeichen zu achten (positiv/negativ) und es auf das Ergebnis anzuwenden.
  • Der Wertebereich eines 32-Bit-Integers liegt zwischen -2147483648 und 2147483647.
  • Denken Sie daran, Division durch Null zu vermeiden oder eine Ausnahme auszulösen.
{% include "before_you_begin.html" %} {% include "how_to_test_head.html" %}
  • Rufen Sie Ihre Funktion in einer Python-Shell oder in Ihrem Programm auf:
    print(divide(10, 3))
    Das Programm sollte ausgeben:
    3
  • Rufen Sie Ihre Funktion mit negativen Zahlen auf:
    print(divide(7, -3))
    Das Programm sollte ausgeben:
    -2
  • Testen Sie die Trunkierung zur Null:
    print(divide(-7, 3))
    Das Programm sollte ausgeben:
    -2
  • Testen Sie die Null-Division:
    divide(10, 0)
  • Testen Sie Grenzwerte:
    print(divide(2147483648, 1))
    Das Programm sollte ausgeben:
    2147483647
  • Und für den kleinsten Wert:
    print(divide(-2147483649, 1))
    Das Programm sollte ausgeben:
    -2147483648
