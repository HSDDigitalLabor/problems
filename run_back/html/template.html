Implementieren Sie den in der Vorlesung vorgestellten Algorithmus zur binären Suche, verwenden Sie diesmal einen rekursiven Ansatz.

{% include "file.html" %} Implementieren Sie die folgende Funktion in Ihrer Datei: def binary_recursive(lst, key):

Die Funktion soll:

  • Eine sortierte Liste lst und einen Suchwert key als Parameter erhalten.
  • lst wird eine sortierte Liste von Zahlen (Integer) sein, in der gesucht wird.
  • key ist eine ganze Zahl (Integer), nach der gesucht wird.
  • Mit Hilfe von Start- (low) und Endindizes (high) sowie einer Mitte (mid) den Suchbereich schrittweise halbieren.
  • Rekursiv den Suchbereich eingrenzen, bis das Element gefunden oder der Bereich leer ist.
  • Den Index des Elements zurückgeben, falls key gefunden wurde.
  • -1 zurückgeben, wenn das Element nicht gefunden wird.
Hinweise
  • Die Liste lst muss sortiert sein, damit der Algorithmus korrekt funktioniert.
  • Verwenden Sie keine eingebauten Suchfunktionen wie index(), find(), count() usw.
  • Verwenden Sie keine lineare Durchläufe der Form for x in lst oder while i < len(lst).
  • Geben Sie nur den Index oder -1 zurück, keine Ausgaben mit print().
{% include "before_you_begin.html" %} {% include "how_to_test_head.html" %}
  • Ihr Programm soll keine Eingabe vom Benutzer verlangen, sondern nur die Funktion definieren. Zum Testen können Sie eine main()-Funktion schreiben, in welcher Sie bestimmte Fälle testen. Dann können Sie mit python {{ filename }} Ihr Programm ausführen.
  • Testen Sie die Funktion zum Beispiel mit: 
    
print(binary_recursive([6, 7, 8, 9, 10], 8))        # Erwartet: 2
print(binary_recursive([11, 13, 15, 17, 18], 17))   # Erwartet: 3
print(binary_recursive([3, 4, 5, 6, 7], 1))         # Erwartet: -1
print(binary_recursive([], 13))                     # Erwartet: -1
