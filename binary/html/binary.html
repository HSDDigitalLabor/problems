Binary Search

Implementieren Sie den in der Vorlesung vorgestellten Algorithmus zur binären Suche. Dieser Algorithmus findet ein Element in einer sortierten Liste deutlich schneller als die lineare Suche, indem er die Liste bei jedem Schritt halbiert.

Erstellen Sie eine Datei mit dem Namen binary.py und Implementieren Sie die folgende Funktion in Ihrer Datei: def binary_search(lst, key):

Die Funktion soll:

  • Eine sortierte Liste lst und einen Suchwert key als Parameter erhalten.
  • lst wird eine sortierte Liste von Zahlen (Integer) sein, in der gesucht wird.
  • key ist eine ganze Zahl (Integer), nach der gesucht wird.
  • Mit Hilfe von Start- (low) und Endindizes (high) sowie einer Mitte (mid) den Suchbereich schrittweise halbieren.
  • Den Index des Elements zurückgeben, falls key gefunden wurde.
  • -1 zurückgeben, wenn das Element nicht gefunden wird.
Hinweise
  • Die Liste lst muss sortiert sein, damit der Algorithmus korrekt funktioniert.
  • Verwenden Sie keine eingebauten Suchfunktionen wie index(), find(), count() usw.
  • Verwenden Sie keine lineare Durchläufe der Form for x in lst oder while i < len(lst).
  • Geben Sie nur den Index oder -1 zurück, keine Ausgaben mit print().

So testen Sie Ihr Programm

So können Sie Ihr Programm manuell testen:

  • Ihr Programm sollte keine Eingabe vom Benutzer verlangen, sondern nur die Funktion definieren. Zum Testen können Sie eine main()-Funktion schreiben, in welcher Sie bestimmte Fälle testen. Dann können Sie mit python binary.py Ihr Programm ausführen.
  • Testen Sie die Funktion zum Beispiel mit: 
    
print(binary_search([1, 2, 3, 4, 5], 3))   # Erwartet: 2
print(binary_search([1, 2, 3, 4, 5], 5))   # Erwartet: 4
print(binary_search([1, 2, 3, 4, 5], 0))   # Erwartet: -1
print(binary_search([], 10))               # Erwartet: -1
