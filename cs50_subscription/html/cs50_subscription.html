CS50 Subscription

Diese Aufgabe dient zur Überprüfung Ihrer Anmeldung zum GIT_SoSe2026 Kurs. Nach erfolgreicher Anmeldung und Ausführung der folgenden Schritte wird im Moodle-Kurs der Status „bestanden" angezeigt. Beachten Sie, dass die Bewertung in Moodle manuell nach Ablauf der Abgabefrist erfolgt.

Bevor Sie beginnen

Melden Sie sich bei cs50.dev an, klicken Sie auf Ihr Terminal und führen Sie cd ohne Parameter aus. Sie sollten feststellen, dass der Prompt Ihres Terminals wie unten aussieht:

$

Als nächstes führen Sie

mkdir cs50_subscription
Fehler beim Kopieren!

aus, um einen Ordner namens cs50_subscription in Ihrem Codespace zu erstellen.

Führen Sie anschließend

cd cs50_subscription
Fehler beim Kopieren!

aus, um in dieses Verzeichnis zu wechseln. Der Prompt Ihres Terminals sollte nun cs50_subscription/$ anzeigen. Jetzt können Sie

wget https://raw.githubusercontent.com/HSDDigitalLabor/problems/refs/heads/git2025/cs50_subscription/files/cs50_subscription.py
Fehler beim Kopieren!

ausführen, um eine Datei namens cs50_subscription.py herunterzuladen, die Sie anschließend zu Testzwecken Abgeben

Abgabe

Führen Sie im Terminal den folgenden Befehl aus, um Ihre Arbeit einzureichen.

submit50 HSDDigitalLabor/problems/git2025/cs50_subscription
Fehler beim Kopieren!

Führen Sie diesen Befehl vor dem Fälligkeitsdatum aus. Sie können Ihre Lösung nach dem ersten Einsenden noch ändern und erneut einreichen. Bewertet wird die zuletzt eingereichte Version vor dem Fälligkeitsdatum. Nach Ablauf der Frist können Sie technisch zwar noch Abgaben tätigen, Ihre Lösung wird jedoch nicht mehr gewertet. Das Ergebnis kann ausschließlich mit obigem Befehl abgegeben werden, eine Abgabe in Moodle ist nicht möglich.
