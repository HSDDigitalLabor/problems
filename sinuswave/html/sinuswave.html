Sinus-Lookup-Table Generator

Eine Look-Up-Tabelle (LUT) für eine Sinusfunktion enthält diskrete Werte einer vollständigen Sinusperiode. Solche Tabellen werden häufig in der digitalen Signalverarbeitung, bei der Klangsynthese oder in eingebetteten Systemen verwendet, um Sinuswerte effizient bereitzustellen.

  • Berechnen Sie für eine vorgegebene Anzahl von Punkten num_points gleichmäßig verteilte Stützstellen über eine vollständige Periode der Sinusfunktion, also im Bereich von 0 bis \(2\pi\).
  • Skalieren Sie die berechneten Sinuswerte aus dem Bereich [-1, 1] in den Bereich [0, MaxAmplitude]. Die Ergebnisse sollen als Ganzzahlen gespeichert werden.
    Beispiel: Bei einer Maximalamplitude von 1023 entspricht der Mittelpunkt der Sinusfunktion ungefähr dem Wert 512.
  • Geben Sie die erzeugten Werte formatiert aus und speichern Sie sie Zeilenweise in der Datei sinus.mem

Erstellen Sie eine Datei mit dem Namen sinuswave.py und implementieren Sie eine Funktion def generateLUT(num_points, max_amplitude):, die eine vollständige Sinus-Look-Up-Tabelle erzeugt.

Die Funktion erhält als Argumente:

  • num_points: Anzahl der Stützstellen einer vollständigen Periode
  • max_amplitude: Maximalwert der skalierten Amplitude

Die Funktion soll die erzeugte Look-Up-Tabelle in der Datei sinus.mem im aktuellen Verzeichnis speichern. Die Argumente sind auf Gültigkeit zu prüfen. Im Fehlerfall soll eine ValueError Exception ausgelöst werden.

Hinweise
  • Verwenden Sie zur Berechnung der Sinuswerte das Modul math, insbesondere die Funktion math.sin().
  • Denken Sie daran, dass Werte passend gerundet und anschließend als Ganzzahlen verarbeitet werden sollen.
  • Prüfen Sie sorgfältig, ob die Eingabewerte sinnvoll sind, z. B. ob die Anzahl der Punkte größer als 0 ist und der Maximalwert der Amplitude positiv ist.
  • Denken Sie daran, Python bietet die Möglichkeit Fehlermeldungen zu generieren, siehe docs.python.org/3/tutorial/errors.html#raising-exceptions .
