Einsteins berühmte Formel

Selbst wenn Sie keinen Physikleistungskurs in der Schule hatten, haben Sie vielleicht schon einmal von der Gleichung E = mc2 gehört. In dieser steht E für Energie (gemessen in Joule), m für Masse (gemessen in Kilogramm) und c für die Lichtgeschwindigkeit (ungefähr 300.000.000 Meter pro Sekunde), laut Albert Einstein et al. Im Wesentlichen bedeutet die Formel, dass Masse und Energie äquivalent sind.

Erstellen Sie eine Datei mit dem Namen einstein.py und schreiben Sie ein Programm in Python, das den Benutzer nach einer Masse als Ganzzahl (in Kilogramm) fragt und dann die entsprechende Anzahl an Joule als Ganzzahl ausgibt. Gehen Sie davon aus, dass der Benutzer eine Ganzzahl (Integer) eingibt.

Hinweise
  • Denken Sie daran, dass input einen str zurückgibt docs.python.org/3/library/functions.html#input.
  • Denken Sie daran, dass die Funktion int einen str in eine int umwandelt docs.python.org/3/library/functions.html#int.
  • Denken Sie daran, dass Python einige built-in Funktionen enthält, siehe docs.python.org/3/library/functions.html.

Bevor Sie beginnen

Melden Sie sich bei cs50.dev an, klicken Sie auf Ihr Terminal und führen Sie cd ohne Parameter aus. Sie sollten feststellen, dass der Prompt Ihres Terminals wie unten aussieht:

Als nächstes führen Sie

aus, um einen Ordner namens einstein in Ihrem Codespace zu erstellen.

Führen Sie anschließend

cd einstein

aus, um in dieses Verzeichnis zu wechseln. Der Prompt ihres Terminals sollte nun einstein/$ anzeigen. Jetzt können Sie

ausführen, um eine Datei namens einstein.py zu erstellen, in der Sie Ihr Programm schreiben.

  • Führen Sie Ihr Programm mit python einstein.py aus. Ihr Programm sollte ausgeben: m: und auf die Eingabe des Benutzers warten. Geben Sie 14 ein und drücken Sie Enter. Ihr Programm sollte ausgeben:

  • Führen Sie Ihr Programm mit python einstein.py aus. Ihr Programm sollte ausgeben: m: und auf die Eingabe des Benutzers warten. Geben Sie 50 ein und drücken Sie Enter. Ihr Programm sollte ausgeben:

Abgabe

Führen Sie im Terminal den folgenden Befehl aus, um Ihre Arbeit einzureichen.

submit50 HSDDigitalLabor/problems/git2025/einstein

Führen Sie diesen Befehl vor dem Fälligkeitsdatum aus. Sie können Ihre Lösung nach dem ersten Einsenden noch ändern und erneut einreichen. Bewertet wird die zuletzt eingereichte Version vor dem Fälligkeitsdatum. Nach Ablauf der Frist können Sie technisch zwar noch Abgaben tätigen, Ihre Lösung wird jedoch nicht mehr gewertet. Das Ergebnis kann ausschließlich mit obigem Befehl abgegeben werden, eine Abgabe in Moodle ist nicht möglich.
