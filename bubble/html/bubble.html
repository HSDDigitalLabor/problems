Bubble-Sort

Der Bubble-Sort-Algorithmus vergleicht wiederholt benachbarte Elemente und vertauscht sie, wenn sie nicht in der gewünschten Reihenfolge sind, bis die Liste vollständig sortiert ist. Dabei "blubbern" die größten oder kleinsten Elemente wie Luftblasen an das Ende der Liste.

Aufgabe

Implementieren Sie die Funktion bubble_sort(lst), die eine Liste von Zahlen mithilfe des Bubble-Sort-Algorithmus in aufsteigender Reihenfolge sortiert. Die Funktion erhält die unsortierte Liste lst als Parameter und gibt die sortierte Liste zurück.

Die Funktionsdefinition ist bereits vorgegeben, verwenden Sie diese: 


def bubble_sort(lst):
    ...
Verwenden sie keine eingebauten Sortierfunktionen wie sorted() oder list.sort().

Hinweise
  • Sie können die Liste mit zwei verschachtelten for-Schleifen durchlaufen. Vergleichen Sie jeweils zwei benachbarte Elemente lst[j] und lst[j + 1] und vertauschen Sie diese, falls sie die falsche Reihenfolge aufweisen.
  • Nach jedem vollständigen Durchlauf wird das größte Element „nach oben gebubbelt“. Daher können Sie in der inneren Schleife bei jedem Durchgang ein Element weniger überprüfen.
  • Eine Verbesserung des Algorithmus besteht darin, den Sortiervorgang vorzeitig abzubrechen, wenn in einem Durchlauf keine Vertauschungen vorgenommen wurden. Hierfür können Sie eine Variable swapped einführen, um zu überprüfen, ob während eines Durchlaufs Vertauschungen stattgefunden haben. Wenn dies nicht der Fall ist, ist die Liste bereits sortiert, und der Algorithmus kann vorzeitig beendet werden.
