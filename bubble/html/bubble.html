Bubblesort

In dieser Übung implementieren Sie die Funktion bubble_sort(lst), die eine Liste von Zahlen mit dem Bubble-Sort-Algorithmus sortiert. Bubble Sort vergleicht benachbarte Elemente und vertauscht sie, wenn sie in der falschen Reihenfolge sind. Dies wiederholt sich so lange, bis die gesamte Liste sortiert ist.

Die Funktion sollte wie folgt aussehen: 

def bubble_sort(lst):
    ...
Die Funktion nimmt eine Liste lst als Parameter und gibt die sortierte Liste zurück. Sie sollen keine eingebauten Sortierfunktionen wie sorted() oder list.sort() verwenden.

Hinweise
  • Sie können die Liste mit zwei verschachtelten for-Schleifen durchlaufen. Vergleichen Sie jeweils zwei benachbarte Elemente lst[j] und lst[j + 1] und vertauschen Sie sie, falls sie in der falschen Reihenfolge stehen.
  • Nach jedem vollständigen Durchlauf wird das größte Element „nach oben gebubbelt“. Sie können daher in der inneren Schleife bei jedem Durchlauf ein Element weniger prüfen.
  • Sie können eine Variable swapped verwenden, um festzustellen, ob in einem Durchlauf Vertauschungen vorgenommen wurden. Wenn keine Vertauschungen stattgefunden haben, ist die Liste bereits sortiert und Sie können abbrechen.

So können Sie Ihr Programm manuell testen:

    Testen Sie die Funktion mit einer kleinen Liste, z. B.:

  • Testen Sie eine bereits sortierte Liste: 
    print(bubble_sort([1, 2, 3]))  # sollte [1, 2, 3] ausgeben
