Compare Strings

Erstellen Sie eine Datei mit dem Namen strings.py und implementieren Sie ein Python-Programm, das den Benutzer nach einer Zeichenkette fragt und prüft, ob diese mit dem String hello world übereinstimmt. Beim Vergleich soll die Groß- und Kleinschreibung ignoriert werden. Geben Sie aus, ob die beiden Strings gleich sind oder nicht.

Hinweise
  • Denken Sie daran, dass input einen str zurückgibt, siehe docs.python.org/3/library/functions.html#input.
  • Denken Sie daran, dass Sie einen str mit print ausgeben können.
  • Denken Sie daran, dass ein str viele Methoden besitzt, siehe docs.python.org/3/library/stdtypes.html#string-methods .

Bevor Sie beginnen

Melden Sie sich bei cs50.dev an, klicken Sie auf Ihr Terminal und führen Sie cd ohne Parameter aus. Sie sollten feststellen, dass der Prompt Ihres Terminals wie unten aussieht:

$

Als nächstes führen Sie

mkdir strings

aus, um einen Ordner namens strings in Ihrem Codespace zu erstellen.

Führen Sie anschließend

cd strings

aus, um in dieses Verzeichnis zu wechseln. Der Prompt ihres Terminals sollte nun strings/$ anzeigen. Jetzt können Sie

code strings.py

ausführen, um eine Datei namens strings.py zu erstellen, in der Sie Ihr Programm schreiben.

Abgabe

Führen Sie im Terminal den folgenden Befehl aus, um Ihre Arbeit einzureichen.

Führen Sie diesen Befehl vor dem Fälligkeitsdatum aus. Sie können Ihre Lösung nach dem ersten Einsenden noch ändern und erneut einreichen. Bewertet wird die zuletzt eingereichte Version vor dem Fälligkeitsdatum. Nach Ablauf der Frist können Sie technisch zwar noch Abgaben tätigen, Ihre Lösung wird jedoch nicht mehr gewertet.
