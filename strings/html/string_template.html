{% include "file.html" %} implementieren Sie ein Python-Programm, das den Benutzer nach einer Zeichenkette fragt und prüft, ob diese mit dem String hello world übereinstimmt. Beim Vergleich soll die Groß- und Kleinschreibung ignoriert werden. Geben Sie aus, ob die beiden Strings gleich sind oder nicht.

Hinweise
  • Denken Sie daran, dass input einen str zurückgibt, siehe docs.python.org/3/library/functions.html#input.
  • Denken Sie daran, dass Sie einen str mit print ausgeben können.
  • Denken Sie daran, dass ein str viele Methoden besitzt, siehe docs.python.org/3/library/stdtypes.html#string-methods .
