Aufgabe

{% include "file.html" %} Implementieren Sie eine Funktion def rotate_list(liste, positionen):, die eine gegebene numerische Liste um eine bestimmte Anzahl von Positionen nach rechts rotiert. Jedes Element der Liste wird dabei um positionen verschoben. Elemente, die über das Ende der Liste hinausgehen, werden am Anfang wieder eingefügt. Die Funktion soll die rotierte Liste zurückgeben. Die Rotation soll "in-place" erfolgen, d.h. es soll keine neue Liste erstellt werden. Zusätzlicher Speicher darf ausschließlich für Hilfsvariablen verwendet werden.

Die Funktionsdefinition ist bereits vorgegeben, verwenden Sie diese: 


def rotate_list(liste, positionen):
    ...
Hinweise
  • Denken Sie daran, das es möglich ist auf einzelne Elemente einer Liste zuzugreifen, siehe https://docs.python.org/3/tutorial/introduction.html#lists .

Beispiele

Beispiel 1


Eingabe: liste = [1,2,3,4,5,6,7], positionen = 3
Ausgabe: [5,6,7,1,2,3,4]

Erklärung:
rotiere um 1 Position nach rechts:   [7,1,2,3,4,5,6]
rotiere um 2 Positionen nach rechts: [6,7,1,2,3,4,5]
rotiere um 3 Positionen nach rechts: [5,6,7,1,2,3,4]

Beispiel 2


Eingabe: liste = [-1,-100,3,99], positionen = 3
Ausgabe: [-100,3,99,-1]

Erklärung:
rotiere um 1 Position nach rechts:   [99,-1,-100,3]
rotiere um 2 Positionen nach rechts: [3,99,-1,-100]
rotiere um 3 Positionen nach rechts: [-100,3,99,-1]
{% include "before_you_begin.html" %} {% include "how_to_test_head.html" %}
  • Einfacher Test (Beispiel): 
    liste = [1,2,3,4,5,6,7]
print(rotate_list(liste, 3))  # sollte [5,6,7,1,2,3,4] ausgeben
  • Rotation um 0 Positionen: 
    liste = [1,2,3]
print(rotate_list(liste, 0))  # sollte [1,2,3] ausgeben
  • Rotation gleich Listenlänge: 
    liste = [1,2,3,4]
print(rotate_list(liste, 4))  # sollte [1,2,3,4] ausgeben
  • Rotation größer als die Listenlänge: 
    liste = [1,2,3]
print(rotate_list(liste, 4))  # sollte [3,1,2] ausgeben (da 4 % 3 == 1)
  • Mit negativen Zahlen: 
    liste = [-1,-100,3,99]
print(rotate_list(liste, 3))  # sollte [-100,3,99,-1] ausgeben
