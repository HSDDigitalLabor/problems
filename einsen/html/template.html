In den letzten beiden Problemen zum Gauß-Algorithmus haben Sie Funktionen implementiert, mit deren Hilfe es gelungen ist, eine Matrix in Zeilenstufenform zu bringen. In diesem Problem werden Sie die Funktionalität erweitern und die Koeffizientenmatrix in die reduzierte Stufenform überführen. Dadurch ist das Identifizieren der Lösungen eines linearen Gleichungssystems noch einfacher möglich (Gauß-Jordan-Algorithmus).

Beschreibung des Algorithmus als Textcode

Algorithmus (Reduzierte Zeilenstufenform):

  • Voraussetzung: Die Matrix ist in Zeilenstufenform. Idealerweise sind alle Pivot-Elemente bereits 1.
  • Bearbeite die Pivotzeilen von unten nach oben.
  • Schritt 1 - Pivot suchen: Finde in Zeile i die Pivotspalte j (erste 1 von links).
  • Schritt 2 - Pivot skalieren: Ist der Pivot nicht 1, verwende die Funktion rowMult, um den Pivot auf 1 zu skalieren.
  • Schritt 3 - Zeilen oberhalb des Pivots eliminieren: Für jede Zeile k oberhalb von i verwende die Funktion: rowAdd damit der Eintrag in Spalte j in Zeile k auf 0 gesetzt wird.
  • Wiederholung: Wiederhole für alle Pivotzeilen.
  • Am Ende sind in jeder Pivotspalte nur noch die Einträge 0 außer einer 1 im Pivot. Damit ist die Matrix in reduzierter Zeilenstufenform.

Aufgabe

Implementieren Sie die Funktion reduzierteStufen in Python. Die Funktion bekommt die Matrix M in Stufenform als Liste von Listen übergeben und soll die Matrix M in-place verändern.

Die äußere Liste repräsentiert die Zeilen, die inneren Listen die Spalten der Matrix. Verwenden Sie die Hilfsfunktionen rowMult und rowAdd aus dem Problem Zeilen. Falls Sie das Problem Zeilen noch nicht gelöst haben, belassen Sie die Funktionsdefinitionen in Ihrer Datei und testen Sie Ihren Code mit check50 (die fehlenden Funktionen werden beim Check ergänzt).

def rowMult(M, i, k):
    """Multipliziert die i-te Zeile der Matrix M mit dem Faktor k."""


def rowAdd(M, i, j, k):
    """Addiert das k-fache der i-ten Zeile zur j-ten Zeile der Matrix M."""


def reduzierteStufen(M):
    """Bringt die Matrix M in reduzierte Stufenform."""

Beachten Sie dabei folgende Anforderungen:

  • Die Matrix M wird als Liste von Listen übergeben (äußere Liste = Zeilen, innere Listen = Spalten).
  • Die Operationen sollen die Matrix in-place verändern (keine neue Matrix zurückgeben).
Hinweise
  • In Python werden List-Objekte als Referenzen übergeben, daher wirken sich Änderungen in den Funktionen direkt auf die ursprüngliche Matrix aus.
  • Wegen Rundungsfehlern bei Gleitkommazahlen ggf. mit geeigneter Toleranz auf 0 prüfen.
