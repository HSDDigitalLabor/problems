Scratch: Binärzähler

Erstellen Sie in Scratch einen Binärzähler, mit dem Sie eine fünfstellige Binärzahl von 00000 bis 11111 eingeben und automatisch in die entsprechende Dezimalzahl umwandeln können.

Die Online Programmierumgebung Scratch finden Sie unter scratch.mit.edu unter "Entwicklen" befindet sich der bekannte Editor. Um Ihre Projekte speichern zu können, ist eine Anmeldung mit Benutzername und Passwort erforderlich. Erstellen Sie sich dazu einen Account durch Drücken des Buttons "Werde Scratcher".

Ist Ihnen der Begriff Binärzähler nicht geläufig, schauen Sie sich bitte folgendes Video an: Binär zählen mit der Hand!

Folgende Schritte sind notwendig:

  • Um den Binärzähler zu erstellen, benötigen Sie fünf identische Figuren, die jeweils ein Kostüm mit den Zahlen 0 und 1 haben. Jede Figur repräsentiert eine Binärstelle (Bit) und kann durch Klicken zwischen den Kostümen 0 und 1 wechseln.
  • Erstellen Sie außerdem eine Variable, die den Wert der Dezimalzahl anzeigt.
  • Beim Klick auf eine Binärstelle soll sich das Kostüm wechseln und die Dezimalzahl entsprechend angepasst werden. Dazu müssen Sie für jede der fünf Figuren ein Skript erstellen und die Dezimalzahl entsprechend berechnen.
  • Beim Neustart des Projekts soll die Dezimalzahl und der Binärzähler auf 0 zurückgesetzt werden.
  • Haben Sie den Binärzähler fertiggestellt, testen Sie ihn, indem Sie verschiedene Binärzahlen eingeben und überprüfen, ob die Dezimalzahl korrekt angezeigt wird.

Nachdem Sie die Aufgabe abgeschlossen haben, überprüfen Sie nochmals den erstellten Code und überlegen Sie, ob es noch Verbesserungsmöglichkeiten gibt. (Beispielsweise Entfernen von doppeltem Code, bessere Benennung von Variablen, etc.)

Gerne können Sie Ihr Programm auch noch erweitern, z.B. durch eine höhere Anzahl an Bit, Sprachanweisungen, etc.
