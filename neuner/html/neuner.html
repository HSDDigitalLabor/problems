Neunerprobe

Die Neunerprobe ist eine Methode zur Überprüfung von Multiplikationsergebnissen. Dazu wird für die Faktoren und das Produkt eine einstellige Prüfsumme gebildet, die anschließend miteinander verglichen werden, um Rechenfehler zu erkennen.

Das Verfahren:

  1. Prüfsumme berechnen:
    • Bilde die Quersumme der Zahl (Summe aller Ziffern)
    • Falls das Ergebnis mehrstellig ist: Bilde erneut die Quersumme
    • Wiederhole dies, bis nur noch eine Ziffer übrig bleibt
  2. Anwendung auf die Multiplikation a · b = c:
    • Berechne die Prüfsumme von a → Ergebnis: PSUMa
    • Berechne die Prüfsumme von b → Ergebnis: PSUMb
    • Berechne die Prüfsumme von c → Ergebnis: PSUMc
  3. Vergleich durchführen:
    • Multipliziere PSUMa · PSUMb
    • Bilde die Prüfsumme dieses Produkts → Ergebnis: PSUMprod
    • Vergleiche PSUMprod mit PSUMc
  4. Auswertung:
    • Falls PSUMprod ≠ PSUMc: Ist das Ergebnis c definitiv falsch
    • Falls PSUMprod = PSUMc: Kann das Ergebnis c korrekt sein (aber nicht garantiert)

Wichtig: Die Methode ist nicht narrensicher – sie erkennt viele, aber nicht alle Fehler. Ein falsches Ergebnis kann den Test bestehen.

Beispiel: a = 429, b = 357, c = 154153

  • PSUMa = 4+2+9 = 15 → 1+5 = 6
  • PSUMb = 3+5+7 = 15 → 1+5 = 6
  • PSUMc = 1+5+4+1+5+3 = 19 → 1+9 = 10 → 1+0 = 1
  • PSUMa · PSUMb = 6 · 6 = 36 → 3+6 = 9
  • Da PSUMprod = 9 ≠ 1 = PSUMc, ist c falsch

Aufgabe

Schreiben Sie eine Funktion isValidProduct(a, b, c), die die Faktoren a, b und das Ergebnis c (als Strings oder Zahlen) entgegennimmt und mittels Neunerprobe prüft, ob c korrekt sein kann. Die Funktion soll True zurückgeben, wenn die Prüfsumme von a · b mit der Prüfsumme von c übereinstimmt, sonst False. Leere oder nicht-numerische Eingaben sind zu behandeln.

Erstellen Sie eine Datei mit dem Namen neuner.py und Implementieren Sie Ihr Programm so, dass die Funktion isValidProduct(a, b, c) die Neunerprobe korrekt durchführt.

Hinweise
  • Die Prüfsumme einer Zahl kann durch wiederholtes Bilden der Quersumme ermittelt werden, bis nur noch eine einstellige Zahl übrig bleibt.
  • Behandeln Sie Sonderfälle: leere Strings, nicht-numerische Eingaben und negative Zahlen.
  • Die Eingaben a, b und c können sowohl als Strings als auch als Zahlen übergeben werden.
  • Die Funktion soll True zurückgeben, wenn die Neunerprobe besteht (also s = sc), sonst False.
  • Die Bildung der einstelligen Prüfsumme (Quersumme bis zur Einerstelle) kann iterativ oder rekursiv implementiert werden. Es ist jedoch nicht erforderlich, Rekursion zu verwenden.
