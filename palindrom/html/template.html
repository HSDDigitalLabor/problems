Im Deutschen gibt es Wörter, die vorwärts und rückwärts gelesen genau das gleiche Wort ergeben, sogenannte Palindrome.

Siehe Wikipedia: Palindrom .

Das Palindrom wird auch gerne besungen wie in diesem Video:

Neben Wortpalindromen und Satzpalindromen können auch Zahlen ein Palindrom darstellen, z.B. 12321 oder 121.
Achtung: eine negative Zahl wie -121 ist kein Palindrom, da das Minuszeichen vorne steht.

Aufgabe

{% include "file.html" %} implementieren Sie eine Funktion def isPalindrome(number):, die Überprüft ob eine Ganzzahl ein Palindrom ist. Die Funktion erhält als Argument eine Ganzzahl int und gibt einen Wahrheitswert bool (True oder False) zurück.

Hinweise
  • Denken Sie daran, dass ein str mit einigen Methoden ausgestattet ist, siehe docs.python.org/3/library/stdtypes.html#string-methods .
  • Denken Sie daran, dass Python einige eingebaute Funktionen hat, siehe docs.python.org/3/library/functions.html .
  • Denken Sie daran, das es möglich ist auf einzelne Teile eines Strings zuzugreifen, siehe docs.python.org/3/tutorial/introduction.html#text .

  • Ihr Programm sollte keine Benutzereingabe verlangen, sondern nur die Funktion definieren. Sie können zum Testen eine main()-Funktion nutzen und Ihr Programm mit python {{filename}} ausführen.
  • Testen Sie Ihre Funktion mit verschiedenen Ganzzahlen, z.B.: 
    
print(isPalindrome(121))   # Erwartet: True
print(isPalindrome(-121))  # Erwartet: False
print(isPalindrome(12321)) # Erwartet: True
print(isPalindrome(-101))  # Erwartet: False

Testen Sie unbedingt auch weitere Zahlen, um sicherzugehen, dass Ihre Funktion korrekt arbeitet!
