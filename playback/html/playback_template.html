Manche Menschen neigen dazu, beim Vortragen sehr schnell zu sprechen, und es wäre schön, sie zu verlangsamen – ähnlich wie bei YouTubes 0,75-facher Wiedergabegeschwindigkeit oder indem sie zwischen den Wörtern pausieren.

{% include "file.html" %} implementieren Sie ein Python-Programm, das den Benutzer nach einer Eingabe fragt und anschließend genau diese Eingabe ausgibt, wobei jedes Leerzeichen durch ... (drei Punkte) ersetzt wird.

Hinweise
  • Denken Sie daran, dass Sie mit input einen str erhalten, siehe docs.python.org/3/library/functions.html#input.
  • Denken Sie daran, dass Sie einen str mit print ausgeben können, siehe docs.python.org/3/library/functions.html#print. Recall that a str comes with quite a few methods, per docs.python.org/3/library/stdtypes.html#string-methods.
