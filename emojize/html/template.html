Da Emojis nicht so einfach zu tippen sind wie Text, zumindest auf Laptops und Desktops, unterstützen einige Programme „Codes“, sodass Sie zum Beispiel :thumbs_up: eingeben können, was automatisch in 👍 umgewandelt wird. Einige Programme unterstützen zusätzlich Aliase, sodass Sie zum Beispiel kürzer :thumbsup: eingeben können, was ebenfalls automatisch in 👍 umgewandelt wird.
Siehe carpedm20.github.io/emoji/all.html?enableList=enable_list_alias für eine Liste von Codes mit Aliassen.

{% include "file.html" %} implementieren Sie ein Programm, das den Benutzer nach einem Text fragt und dann die „emojisierte“ Version dieser Zeichenkette ausgibt, alle im Eingabetext enthaltenen Codes (oder Aliasse) werden in das entsprechende Emoji umgewandelt.

Hinweise
  • Beachten Sie, dass das emoji Modul laut pypi.org/project/emoji zwei Funktionen enthält, eine davon ist emojize. Diese Funktion akzeptiert einen optionalen, benannten Parameter namens language. Sie können das Modul mit folgendem Befehl installieren:
    pip install emoji
