Aufgabe

Implementieren Sie die Funktion merge(intervals), die eine Liste von Intervallen zusammenführt. Überlappende Intervalle werden dabei zu einem einzigen Intervall kombiniert. Die Funktion erhält als Parameter eine Liste intervals, die mehrere Intervalle enthält. Jedes Intervall wird dabei selbst als eine Liste dargestellt.

Jedes Intervall wird als Liste von zwei Ganzzahlen dargestellt: [start, end] mit start <= end. Die Funktion gibt eine neue Liste der zusammengeführten Intervalle zurück. In dieser Ergebnisliste sind alle Intervalle aufsteigend nach ihrem Startwert start angeordnet.

Die Funktionsdefinition ist bereits vorgegeben, verwenden Sie diese: 


def merge(intervals):
   ...

Verwenden sie keine eingebauten Sortierfunktionen wie sorted() oder list.sort().

Beispiel


Eingabe:  [[1, 3], [2, 6], [8, 10], [15, 18]]
Ausgabe: [[1, 6], [8, 10], [15, 18]]

Eingabe:  [[1, 4], [4, 5]]
Ausgabe: [[1, 5]]
Hinweise
  • Falls sich zwei Intervalle überlappen oder direkt aneinandergrenzen, sollen sie zu einem größeren Intervall zusammengeführt werden.
  • Implementieren Sie das Sortieren der Intervalle selbst, z. B. mit einem einfachen Sortierverfahren wie Bubble Sort oder Insertion Sort.
  • Stellen Sie sicher, dass die Liste intervals nach Ihrem eigenen Sortieralgorithmus strikt aufsteigend nach den jeweiligen start-Werten geordnet ist, bevor Sie mit dem Zusammenführen beginnen. Prüfen Sie dies ggf. dadurch, dass für jedes benachbarte Paar gilt: intervals[i][0] <= intervals[i + 1][0].

Pseudocode


function merge(intervals):
    # wenn Intervalle leere Liste, return leere Liste

    # 1. Sortiere die Intervalle nach ihrem Startwert (manuell)
    # Implementieren Sie dazu einen der vorher besprochenen Sortieralgorithmen

    # 2. Führe überlappende Intervalle zusammen
    merged = empty list
    for interval in intervals:
        if merged is empty:
            append a copy of interval to merged
        else:
            last <-- letztes Element von merged
            if interval.start > last.end:
                append interval to merged
            else:
                last.end = max(last.end, interval.end)

    return merged

  • Testen Sie mit einfachen Intervallen: 
    print(merge([[1, 4], [4, 5]]))  # sollte [[1, 5]] ausgeben
  • Testen Sie mit unsortierten Intervallen: 
    print(merge([[4, 7], [1, 4]]))  # sollte [[1, 7]] ausgeben
  • Testen Sie mit nicht überlappenden Intervallen: 
    print(merge([[1, 2], [3, 4], [5, 6]]))  # sollte [[1, 2], [3, 4], [5, 6]] ausgeben
