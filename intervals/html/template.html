In dieser Übung implementieren Sie die Funktion merge(intervals), die eine Liste von Intervallen zusammenführt, sodass sich überlappende Intervalle zu einem einzigen Intervall kombiniert werden.

Jedes Intervall ist eine Liste aus zwei Ganzzahlen [start, end], wobei start <= end gilt. Die Funktion soll eine neue Liste mit den zusammengeführten Intervallen zurückgeben, wobei die Intervalle nach ihrem Startwert sortiert sind. 

def merge(intervals):
    ...

Beispiel: 


Eingabe:  [[1, 3], [2, 6], [8, 10], [15, 18]]
Ausgabe: [[1, 6], [8, 10], [15, 18]]

Eingabe:  [[1, 4], [4, 5]]
Ausgabe: [[1, 5]]
Hinweise
  • Falls sich zwei Intervalle überlappen oder direkt aneinandergrenzen, sollen sie zu einem größeren Intervall zusammengeführt werden.
  • Sie dürfen keine eingebauten Sortierfunktionen wie sorted() oder list.sort() verwenden. Implementieren Sie das Sortieren der Intervalle selbst, z. B. mit einem einfachen Sortierverfahren wie Bubble Sort oder Insertion Sort.
  • Achten Sie darauf, dass die Eingabeliste intervals nach dem Sortieren alle Intervalle in aufsteigender Reihenfolge ihrer Startwerte enthält.

Pseudocode


function merge(intervals):
    # wenn Intervalle leere List, return leere Liste

    # 1. Sortiere die Intervalle nach ihrem Startwert (manuell)
    # Implementieren Sie dazu einen der vorher besprochen Sortieralgorithmus

    # 2. Führe überlappende Intervalle zusammen
    merged = empty list
    for interval in intervals:
        if merged is empty:
            append a copy of interval to merged
        else:
            last <-- letztes Element von merged
            if interval.start > last.end:
                append interval to merged
            else:
                last.end = max(last.end, interval.end)

    return merged
