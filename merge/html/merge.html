Merge

In dieser Übung implementieren Sie die Funktion merge(lst1, lst2), die zwei sortierte Listen zu einer neuen, ebenfalls sortierten Liste zusammenführt. Die ursprünglichen Listen sollen dabei unverändert bleiben.

Die Funktion sollte wie folgt aussehen: 

def merge(lst1, lst2):
    ...
Die Funktion nimmt zwei list-Objekte lst1 und lst2 als Parameter entgegen und gibt eine neue Liste zurück, die alle Elemente beider Listen in aufsteigender Reihenfolge enthält.

Hinweise
  • Beide Eingabelisten lst1 und lst2 sind bereits sortiert. Ihre Aufgabe ist es, die Elemente effizient zusammenzuführen – ähnlich wie im Merge-Schritt von Mergesort.
  • Sie dürfen keine eingebauten Sortierfunktionen wie sorted() oder list.sort() verwenden.
  • Die Funktion soll eine zusammengeführte Liste zurückgeben.

So testen Sie Ihr Programm

So können Sie Ihr Programm manuell testen:

  • Testen Sie sie mit einfachen Listen, z.B.: 
    print(merge([1, 3, 5], [2, 4, 6]))  # sollte [1, 2, 3, 4, 5, 6] ausgeben
  • Testen Sie mit einer leeren Liste: 
    print(merge([], [1, 2, 3]))  # sollte [1, 2, 3] ausgeben
  • Testen Sie mit Listen, die doppelte Werte enthalten: 
    print(merge([1, 2, 2], [2, 3]))  # sollte [1, 2, 2, 2, 3] ausgeben
  • Überlegen Sie sich weitere Testfälle und Randbedingungen zum Testen.
