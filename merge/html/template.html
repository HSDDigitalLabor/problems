Merge Sort ist ein effizienter Sortieralgorithmus, der das Prinzip "Teile und Herrsche" anwendet, indem er eine Liste wiederholt in zwei Hälften teilt, diese rekursiv sortiert und anschließend die sortierten Teillisten wieder zu einer einzigen sortierten Liste zusammenführt. Das Zusammenführen der Listen ist dabei der entscheidende Schritt, dieser wird als "Merge" bezeichnet und in diesem Problem genauer betrachtet.

Aufgabe

Implementieren Sie die Funktion merge(lst1, lst2), die zwei aufsteigend sortierte Listen zu einer neuen, ebenfalls aufsteigend sortierten Liste zusammenführt. Die ursprünglichen Listen sollen dabei unverändert bleiben. Die Funktion nimmt zwei list-Objekte lst1 und lst2 als Parameter entgegen und gibt eine neue Liste zurück, die alle Elemente beider Listen in aufsteigender Reihenfolge enthält.

Die Funktionsdefinition ist bereits vorgegeben, verwenden Sie diese: 


def merge(lst1, lst2):
    ...

Verwenden sie keine eingebauten Sortierfunktionen wie sorted() oder list.sort().

Hinweise
  • Beide Eingabelisten lst1 und lst2 sind bereits sortiert. Ihre Aufgabe ist es, die Elemente effizient zusammenzuführen – ähnlich wie im Merge-Schritt von Mergesort.
{% include "before_you_begin.html" %} {% include "how_to_test_head.html" %}
  • Testen Sie sie mit einfachen Listen, z.B.: 
    print(merge([1, 3, 5], [2, 4, 6]))  # sollte [1, 2, 3, 4, 5, 6] ausgeben
  • Testen Sie mit einer leeren Liste: 
    print(merge([], [1, 2, 3]))  # sollte [1, 2, 3] ausgeben
  • Testen Sie mit Listen, die doppelte Werte enthalten: 
    print(merge([1, 2, 2], [2, 3]))  # sollte [1, 2, 2, 2, 3] ausgeben
  • Überlegen Sie sich weitere Testfälle und Randbedingungen zum Testen.
