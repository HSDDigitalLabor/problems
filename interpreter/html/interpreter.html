Math Interpreter

In dieser Aufgabe programmieren Sie einen simplen Math Interpreter, welcher es Nutzern ermöglicht mathematische Ausdrücke auszurechnen.

Aufgabe

Erstellen Sie eine Datei mit dem Namen interpreter.py und Implementieren Sie in der Datei ein Programm, welches den Nutzer nach einem arithmetischen Ausdruck bittet, diesen dann berechnet und das Ergebnis als float-Zahl mit einer Nachkommastelle ausgibt. Nehmen Sie an, dass die Eingabe des Nutzers wie folgt formatiert ist x y z, mit einem Leerzeichen zwischen x und y und einem Leerzeichen zwischen y und z. Dabei ist:

  • x ist ein integer
  • y ist +, -,/ oder *
  • z ist ein integer

Hinweise
  • Denken Sie daran, dass ein str mit einigen Methoden ausgestattet ist, siehe docs.python.org/3/library/stdtypes.html#string-methods .

Bevor Sie beginnen

Melden Sie sich bei cs50.dev an, klicken Sie auf Ihr Terminal und führen Sie cd ohne Parameter aus. Sie sollten feststellen, dass der Prompt Ihres Terminals wie unten aussieht:

$

Als nächstes führen Sie

mkdir interpreter

aus, um einen Ordner namens interpreter in Ihrem Codespace zu erstellen.

Führen Sie anschließend

cd interpreter

aus, um in dieses Verzeichnis zu wechseln. Der Prompt ihres Terminals sollte nun interpreter/$ anzeigen. Jetzt können Sie

code interpreter.py

ausführen, um eine Datei namens interpreter.py zu erstellen, in der Sie Ihr Programm schreiben.

So testen Sie Ihr Programm

So können Sie Ihr Programm manuell testen:

  • Führen Sie Ihr Programm mit python interpreter.py aus. Geben Sie 1 + 1 ein und drücken Sie Enter. Ihr Programm sollte ausgeben:
    2.0
  • Führen Sie Ihr Programm mit python interpreter.py aus. Geben Sie 2 - 3 ein und drücken Sie Enter. Ihr Programm sollte ausgeben:
    -1.0
  • Führen Sie Ihr Programm mit python interpreter.py aus. Geben Sie 2 * 2 ein und drücken Sie Enter. Ihr Programm sollte ausgeben:
    4.0
  • Führen Sie Ihr Programm mit python interpreter.py aus. Geben Sie 50 / 5 ein und drücken Sie Enter. Ihr Programm sollte ausgeben:
    10.0

Sie können das folgende Kommando ausführen, um Ihren Code mit check50 zu überprüfen, einem Programm, das bei der Abgabe verwendet wird um Ihren Code zu testen. Testen Sie Ihr Programm aber auch selbst!
