Implementieren Sie eine Funktion, die in einer sortierten Liste die Einfügeposition für einen gegebenen Wert findet. Wenn der Wert bereits existiert, soll der Index dieses Elements zurückgegeben werden. Andernfalls soll die Position zurückgegeben werden, an der der Wert eingefügt werden müsste, damit die Liste weiterhin sortiert bleibt.

{% include "file.html" %} Implementieren Sie die folgende Funktion in Ihrer Datei: def searchInsert(lst, key):

Die Funktion soll:

  • Eine sortierte Liste lst und einen Suchwert key als Parameter erhalten.
  • Den Index zurückgeben, an welchem key zu finden ist oder eingefügt werden müsste.
  • Effizient arbeiten (d. h. nicht die gesamte Liste linear durchsuchen).
Hinweise
  • Die Liste lst ist sortiert.
  • Verwenden Sie keine eingebauten Suchfunktionen wie index(), find(), count() usw.
  • lst wird eine sortierte Liste von Zahlen (Integer) sein, in der gesucht wird.
  • key ist eine ganze Zahl (Integer), nach der gesucht wird.
  • Vermeiden Sie eine lineare Suche wie for x in lst oder while i < len(lst).
  • Nutzen Sie stattdessen einen Ansatz basierend auf der binären Suche.
  • Geben Sie nur den Index zurück, keine Ausgaben mit print().
