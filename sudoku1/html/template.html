Beim Lösen eines Sudoku-Rätsels ist es wichtig sicherzustellen, dass das Sudoku-Feld gültig ist. Ein gültiges Sudoku erfüllt zwei Bedingungen:

  • In jeder Zeile dürfen die Ziffern 1-9 höchstens einmal vorkommen.
  • In jeder Spalte dürfen die Ziffern 1-9 höchstens einmal vorkommen.

Ihre Aufgabe ist es, eine Funktion isValid(board) zu implementieren, die ein 9x9-Sudoku-Board (als Liste von Listen mit Strings) entgegennimmt und True zurückgibt, wenn das Board die Regeln für Zeilen und Spalten erfüllt, andernfalls False. Leere Felder sind durch "." dargestellt.

{% include "file.html" %} Implementieren Sie Ihr Programm so, dass die Funktion isValid(board) korrekt überprüft, ob das Sudoku-Feld Zeilen und Spalten korrekt befüllt hat.

Hinweise
  • Verwenden Sie keine Funktionen wie .count() oder .index(). Arbeiten Sie stattdessen mit Schleifen und Vergleichen.
  • Leere Felder (".") sollten ignoriert werden.
  • Jedes Element des Boards ist ein String, nicht eine Zahl.
{% include "before_you_begin.html" %} {% include "how_to_test_head.html" %}
  • Ihr Programm wird nicht über Eingaben getestet, sondern ausschließlich über Aufrufe der Funktion isValid mit verschiedenen Sudoku-Boards.
  • Beispiel für ein gültiges Board (Ihre Funktion sollte True zurückgeben):
    [ ["5","3",".",".","7",".",".",".","."], ["6",".",".","1","9","5",".",".","."], [".","9","8",".",".",".",".","6","."], ["8",".",".",".","6",".",".",".","3"], ["4",".",".","8",".","3",".",".","1"], ["7",".",".",".","2",".",".",".","6"], [".","6",".",".",".",".","2","8","."], [".",".",".","4","1","9",".",".","5"], [".",".",".",".","8",".",".","7","9"] ]
  • Beispiel für ein ungültiges Board (Duplikat in einer Zeile):
    [ ["8","8",".",".","7",".",".",".","."], ... ] → Erwartet: False
  • Beispiel für ein ungültiges Board (Duplikat in einer Spalte):
    [ ["5","3",".",".","7",".",".",".","."], ["6",".",".","1","9","5",".",".","."], ["5","9","8",".",".",".",".","6","."], ... ] → Erwartet: False
