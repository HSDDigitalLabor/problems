Erstellen Sie sich ein GitHub-Benutzerkonto unter Verwendung Ihrer HSD-Email Adresse.  

Eine Anleitung zur Erstellung dieses Benutzerkontos (User account) finden Sie unter:
https://docs.github.com/de/get-started/start-your-journey/creating-an-account-on-github

Nachdem Sie Ihr GitHub-Benutzerkonto erfolgreich erstellt haben, tragen Sie Ihren gewählten GitHub Benutzernamen in Moodle ein. Dazu drücken Sie den Button und geben Ihren GitHub Benutzernamen in das sich öffnende Textfeld ein. Anschließend geben Sie die Aufgabe durch Klick auf den Button ab. Zur finalen Einsendung bestätigen Sie die Abgabe nochmals in dem folgenden Dialog.

Hinweis: Dieser Schritt ist notwendig, um eine automatische Bewertung Ihrer gelösten Programmieraufgaben vorzunehmen. Achten Sie auf die korrekte Schreibweise Ihres Benutzernamens, ansonsten ist eine Bewertung Ihrer Aufgaben nicht möglich.
