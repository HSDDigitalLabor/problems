{% include "file.html" %} Implementieren Sie eine Python-Funktion str_index(haystack, needle), die den Index des ersten Vorkommens von needle in haystack zurückgibt. Wenn needle nicht in haystack vorkommt, soll die Funktion -1 zurückgeben.

Hinweise
  • haystack und needle sind beides Zeichenketten (str).
  • Verwenden Sie keine eingebauten Funktionen wie str.find() oder den Operator in.
  • Nutzen Sie stattdessen Schleifen und String-Slicing, um zu prüfen, ob needle ab einer bestimmten Position in haystack vorkommt.
  • Denken Sie daran, dass Sie die Schleife nur bis zu der Position durchlaufen müssen, an der needle noch vollständig in haystack passt.
  • Wenn needle leer ist, geben Sie 0 zurück.
{% include "before_you_begin.html" %} {% include "how_to_test_head.html" %}
  • Testen Sie Ihre Funktion mit einer einfachen Übereinstimmung:
    print(str_index("sadbutsad", "sad"))
    Das Programm sollte ausgeben:
    0
  • Wenn das gesuchte Wort nicht vorkommt:
    print(str_index("leetcode", "leeto"))
    Das Programm sollte ausgeben:
    -1
  • Testen Sie ein Vorkommen in der Mitte:
    print(str_index("mississippi", "issip"))
    Das Programm sollte ausgeben:
    4
  • Testen Sie den Fall, wenn needle leer ist:
    print(str_index("abc", ""))
    Das Programm sollte ausgeben:
    0
  • Testen Sie, wenn haystack und needle gleich sind:
    print(str_index("abc", "abc"))
    Das Programm sollte ausgeben:
    0
