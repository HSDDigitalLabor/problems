Der Name „Bogo-Sort“ setzt sich aus den englischen Wörtern „bogus“ (falsch/unsinnig) und „sort“ (sortieren) zusammen. Er beschreibt treffend die absurde und extrem ineffiziente Idee hinter dem Verfahren: Die Liste wird so lange zufällig permutiert (gemischt), bis sie vollständig sortiert ist. Aufgrund seiner schlechten Laufzeit ist der Algorithmus auch als „Stupid Sort“ bekannt.

Aufgabe

Implementieren Sie die Funktion bogo_sort(lst), die eine Liste lst von Zahlen als Argument entgegennimmt und diese mittels Bogo-Sort sortiert und die sortierte Liste zurückgibt. Implementieren Sie zur Umsetzung eine Hilfsfunktion is_sorted(lst), die prüft, ob die Liste bereits sortiert ist.

Die Funktionen sollten wie folgt aussehen: 


def is_sorted(lst):
    ...


def bogo_sort(lst):
    ...
Verwenden Sie keine eingebauten Sortierfunktionen wie sorted() oder list.sort().

Hinweise
  • Um die Liste zufällig zu mischen können Sie folgende Funktion verwenden https://docs.python.org/3/library/random.html#random.shuffle .
  • Bogo-Sort ist sehr ineffizient. Verwenden Sie daher in Tests nur kurze Listen, z.B. 3-5 Elemente, um eine endlose Bearbeitungszeiten zu vermeiden.
{% include "before_you_begin.html" %} {% include "how_to_test_head.html" %}
  • Testen Sie die Funktion mit einer kleinen Liste, z. B.: 
    print(bogo_sort([3, 2, 1]))  # sollte [1, 2, 3] ausgeben
  • Testen Sie eine bereits sortierte Liste: 
    print(bogo_sort([1, 2, 3]))  # sollte [1, 2, 3] ausgeben
  • Testen Sie eine Liste mit doppelten Werten: 
    print(bogo_sort([3, 1, 3, 2]))  # sollte [1, 2, 3, 3] ausgeben
