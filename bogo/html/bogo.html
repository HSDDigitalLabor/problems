Bogosort

In dieser Übung implementieren Sie die Funktion bogo_sort(lst), die eine Liste von Zahlen mittels Bogosort sortiert. Bogosort funktioniert, indem die Liste so lange zufällig gemischt wird, bis sie vollständig sortiert ist.

Die Funktion sollte wie folgt aussehen: 

def bogo_sort(lst):
    ...
Die Funktion nimmt eine Liste lst als Parameter und gibt die sortierte Liste zurück. Sie sollen keine eingebauten Sortierfunktionen wie sorted() oder list.sort() verwenden.

Hinweise
  • Um die Liste zufällig zu mischen können Sie folgende Funktion verwenden https://docs.python.org/3/library/random.html#random.shuffle .
  • Implementieren Sie zuerst eine Hilfsfunktion is_sorted(lst), die prüft, ob die Liste bereits sortiert ist: 
    def is_sorted(lst):
    Verwenden Sie diese Funktion in Ihrem bogo_sort-Algorithmus.
  • Bogosort ist sehr ineffizient. Verwenden Sie daher in Tests nur kurze Listen, z.B. 3-5 Elemente, um endlose Schleifen zu vermeiden.

So testen Sie Ihr Programm

So können Sie Ihr Programm manuell testen:

  • Importieren Sie Ihre Funktion in Python: 
    from bogo import bogo_sort
    Testen Sie die Funktion mit einer kleinen Liste, z. B.: 
    print(bogo_sort([3, 2, 1]))  # sollte [1, 2, 3] ausgeben
  • Testen Sie eine bereits sortierte Liste: 
    print(bogo_sort([1, 2, 3]))  # sollte [1, 2, 3] ausgeben
  • Testen Sie eine Liste mit doppelten Werten: 
    print(bogo_sort([3, 1, 3, 2]))  # sollte [1, 2, 3, 3] ausgeben
