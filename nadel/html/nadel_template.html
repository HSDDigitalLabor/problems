Die Redewendung "eine Nadel im Heuhaufen suchen" wird verwendet, um eine Aufgabe zu beschreiben, die extrem schwierig oder nahezu unmöglich zu lösen ist. Es betont die Herausforderung, etwas sehr Kleines oder Unauffälliges in einer großen Menge zu finden. Zur Lösung dieser Aufgabe muss manchmal das ganze Dorf zusammenrücken

Aufgabe

{% include "file.html" %} Implementieren Sie eine Python-Funktion str_index(heuhaufen, nadel), die den Index des ersten Vorkommens von nadel in heuhaufen zurückgibt. Wenn nadel nicht in heuhaufen vorkommt, soll die Funktion -1 zurückgeben. Verwenden Sie keine eingebauten Funktionen wie str.find() oder den Operator in.

Hinweise
  • heuhaufen und nadel sind beides Zeichenketten (str).
  • Nutzen Sie Schleifen und String-Slicing, um zu prüfen, ob nadel ab einer bestimmten Position in heuhaufen vorkommt.
  • Denken Sie daran, dass Sie die Schleife nur bis zu der Position durchlaufen müssen, an der nadel noch vollständig in heuhaufen passt.
  • Wenn nadel leer ist, geben Sie 0 zurück.
{% include "before_you_begin.html" %} {% include "how_to_test_head.html" %}
  • Testen Sie Ihre Funktion mit einer einfachen Übereinstimmung:
    print(str_index("sadbutsad", "sad"))
    Das Programm sollte ausgeben:
    0
  • Wenn das gesuchte Wort nicht vorkommt:
    print(str_index("leetcode", "leeto"))
    Das Programm sollte ausgeben:
    -1
  • Testen Sie ein Vorkommen in der Mitte:
    print(str_index("mississippi", "issip"))
    Das Programm sollte ausgeben:
    4
  • Testen Sie den Fall, wenn nadel leer ist:
    print(str_index("abc", ""))
    Das Programm sollte ausgeben:
    0
  • Testen Sie, wenn heuhaufen und nadel gleich sind:
    print(str_index("abc", "abc"))
    Das Programm sollte ausgeben:
    0
