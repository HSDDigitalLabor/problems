Bei der Lösung von linearen Gleichungssystemen mit dem Gauß-Algorithmus werden die Gleichungen in eine Matrixform gebracht. Zur Lösung dieses Problems haben Sie im Problem Zeilen bereits Funktionen für Zeilenoperationen implementiert. In diesem Problem sollen Sie mithilfe dieser Funktionen die Matrix in die Zeilenstufenform bringen.

Beschreibung des Algorithmus als Textcode

Algorithmus:

  • Bearbeite die Matrix von oben nach unten und von links nach rechts.
  • Schritt 1 – Pivot suchen: Finde in der aktuellen Spalte eine Zeile mit einem von Null verschiedenen Eintrag (Pivot).
    • Steht an der aktuellen Position bereits ein Pivot, fahre mit dem nächsten Schritt fort.
    • Steht an der aktuellen Position eine Null, suche in den darunterliegenden Zeilen nach einem von Null verschiedenen Eintrag. Falls eine solche Zeile gefunden wird, verwende die Funktion rowAdd, um diese Zeile zur aktuellen Zeile zu addieren.
  • Schritt 2 – Zeilen unter dem Pivot eliminieren: Verwende die Funktion rowAdd, um alle Einträge unterhalb des Pivots in der aktuellen Spalte auf Null zu setzen.
  • Schritt 3 – Pivot skalieren: Verwende die Funktion rowMult, um den Pivot auf 1 zu skalieren.
  • Schritt 4 – Fortfahren: Setze die Bearbeitung in der nächsten Zeile und der nächsten Spalte fort, bis alle Zeilen oder alle Spalten bearbeitet wurden.

Aufgabe

Implementieren Sie die Funktion zeilenStufen in Python. Die Funktion bekommt die Matrix M als Liste von Listen übergeben und soll die Matrix M in-place verändern.

Die äußere Liste repräsentiert die Zeilen, die inneren Listen die Spalten der Matrix. Verwenden Sie die Hilfsfunktionen rowMult und rowAdd aus dem Problem Zeilen. Falls Sie das Problem Zeilen noch nicht gelöst haben, belassen Sie die Funktionsdefinitionen in Ihrer Datei und testen Sie Ihren Code mit check50 (die fehlenden Funktionen werden beim Check ergänzt).

Sollten Sie das Problem Zeilen nicht gelöst haben, belassen Sie die Funktionsdefinitionen in Ihrer Datei und implementieren Sie nur die Funktion zeilenStufen unter Verwendung der Hilfsfunktionen. Kopieren Sie dazu die folgenden Funktionsdefinitionen in Ihre Datei: 

def rowMult(M, i, k):
    """Multipliziert die i-te Zeile der Matrix M mit dem Faktor k."""


def rowAdd(M, i, j, k):
    """Addiert das k-fache der i-ten Zeile zur j-ten Zeile der Matrix M."""


def zeilenStufen(M):
    """Bringt die Matrix M in Zeilenstufenform."""

Beachten Sie dabei folgende Anforderungen:

  • Die Matrix M wird als Liste von Listen übergeben (äußere Liste = Zeilen, innere Listen = Spalten).
  • Die Operationen sollen die Matrix in-place verändern (keine neue Matrix zurückgeben).
Hinweise
  • In Python werden List-Objekte als Referenzen übergeben, daher wirken sich Änderungen in den Funktionen direkt auf die ursprüngliche Matrix aus.
