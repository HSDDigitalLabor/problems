Linear Search

Implementieren Sie den in der Vorlesung vorgestellten Algorithmus zur linearen Suche. Schreiben Sie dafür eine eigene Funktion, ohne eingebaute Suchfunktionen von Python zu verwenden.

Erstellen Sie eine Datei mit dem Namen linear.py und Implementieren Sie die folgende Funktion in Ihrer Datei: def linear_search(lst, key):

Die Funktion soll:

  • Eine Liste lst und einen Suchwert key als Parameter erhalten.
  • lst wird eine Liste von Zahlen (Integer) sein, in der gesucht wird.
  • key ist eine ganze Zahl (Integer), nach der gesucht wird.
  • Die Liste von Anfang bis Ende durchsuchen.
  • Den Index des ersten Elements zurückgeben, das gleich key ist.
  • -1 zurückgeben, wenn das Element nicht gefunden wird.
Hinweise
  • Verwenden Sie keine eingebauten Suchfunktionen wie index(), find(), count() usw.
  • Verwenden Sie stattdessen eine einfache for oder while-Schleife, um alle Elemente durchzugehen.
  • Geben Sie nur den Index oder -1 zurück, keine Ausgaben mit print().
