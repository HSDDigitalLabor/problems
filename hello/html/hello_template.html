{% include "file.html" %} implementieren Sie ein Python-Programm, das den Benutzer nach seinem Namen fragt und anschließend Hallo, so-und-so ausgibt, wobei so-und-so durch den eingegebenen Namen ersetzt wird.

Hinweise
  • Denken Sie daran, dass Sie mit get_string aus der cs50-Bibliothek einen str vom Benutzer abfragen können.
  • Denken Sie daran, dass Sie einen str mit print ausgeben können.
  • Denken Sie daran, dass Sie in Python formatierte Strings erstellen können, indem Sie dem String ein f voranstellen. Zum Beispiel ersetzt f"{name}" den Wert der Variablen name an der Stelle, an der Sie {name} geschrieben haben.
