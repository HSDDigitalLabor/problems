Die Berechnung der IBAN
Eine deutsche IBAN wird wie folgt aus Kontonummer und Bankleitzahl berechnet:

  • Erstellen Sie eine 24-stellige Zahl nach folgendem Schema: Beginnen Sie mit der Bankleitzahl (achtstellige Zahlenfolge), fügen Sie die auf zehn Stellen aufgefüllte Kontonummer hinzu (ergänzen Sie ggf. führende Nullen) und hängen Sie abschließend die Ziffernfolge 131400 an. Diese Ziffernfolge repräsentiert "DE00", wobei Buchstaben in zweistellige Zahlen umgewandelt werden (A=10, B=11, usw.).
    Beispiel: Für die Kontonummer 123456789 und die Bankleitzahl 50010517 ergibt sich die Zahl 500105170123456789131400.
  • Berechnen Sie den Rest, der bei der Division dieser 24-stelligen Zahl durch 97 entsteht. Ziehen Sie diesen Rest von 98 ab. Das Ergebnis sind die Prüfziffern. Falls das Ergebnis einstellig ist, ergänzen Sie eine führende Null, sodass die Prüfziffer zweistellig ist.
    Beispiel: Die Zahl 500105170123456789131400 ergibt bei Division durch 97 den Rest 57. Rechnung: 98 - 57 = 41. Die Prüfziffer lautet somit 41.
  • Die IBAN erhält man schließlich, indem man die beiden Prüfziffern an die Länderkennung DE anhängt, gefolgt von der Bankleitzahl und der ggf. auf zehn Stellen erweiterten Kontonummer.
    Beispiel: Für die Kontonummer 123456789 und die Bankleitzahl 50010517 ergibt sich die IBAN DE41500105170123456789.
Die Berechnung der Prüfziffern erfolgt für andere Länder nach dem gleichen Verfahren. Jedoch unterscheiden sich die Anordnung von Kontonummer und Bankcode innerhalb der IBAN sowie die Gesamtlänge der IBAN von Land zu Land. Innerhalb eines Landes ist die IBAN-Länge jedoch stets gleich.

Aufgabe

{% include "file.html" %} implementieren Sie eine Funktion def calcCheckDigitsDE(blz, konto):, die die Prüfziffern für eine deutsche IBAN berechnet. Die Funktion erhält als Argumente die Bankleitzahl blz und die Kontonummer konto und gibt die Prüfziffer als int Wert zurück. Die Funktion soll die Argumente auf Gültigkeit prüfen und im Fehlerfall eine ValueError Exception auslösen. Konzipieren Sie die Funktion ausschließlich für deutsche IBANs, d.h. die Länderkennung "DE" wird nur intern zur Berechnung der Prüfziffer benötigt und muss nicht von der Funktion als Argument übernommen werden.

  • Denken Sie daran, dass ein str mit einigen Methoden ausgestattet ist, siehe docs.python.org/3/library/stdtypes.html#string-methods .
  • Denken Sie daran, Python bietet die Möglichkeit Fehlermeldungen zu generieren, siehe docs.python.org/3/tutorial/errors.html#raising-exceptions .
  • Denken Sie daran, in Python können boolesche Werte mit speziellen Operatoren verknüpft werden, siehe docs.python.org/3/library/stdtypes.html#boolean-operations-and-or-not .
